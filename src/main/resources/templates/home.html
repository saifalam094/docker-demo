Welcome to Employee Management App

This application allows you to manage employee records efficiently. You can create, view, update, and delete employee information, including name, role, phone number, and description.

  • Beautiful web interface with Thymeleaf
  • RESTful API for integration
  • CRUD operations for employees
  • MySQL database integration
  • Docker & Docker Compose support
